A tag usada para inserir links em HTML é a tag A. E dentro do href coloca-se a url para onde você quer apontar.

Porém feito somente dessa maneira, a nova página vai sobrepor à sua própria página, por isso é altamente recomendável que você configure o link para abrir em uma nova aba. Veja abaixo

Veja o canal no YouTube do Curso em Vídeo.

Usando links internos

Esta é a página principal do meu site, mas você pode saber muito mais acessando a página 2.

Definindo atributos "rel" dos links

Nofollow = Não seguir, não transmitir autoridade do meu site para o site onde estou apontando
Dofollow = Endosso o site e transmito uma parte da autoridade do meu site para este link (link juice)
Sponsored = Tag usada para links para conteúdo patrocinado e pago
UGC = Tag usada para links para conteúdo gerado pelo usuário: comentários, avaliações, posts etc.
