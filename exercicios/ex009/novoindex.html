Código-fonte e Pré-formatação

O comando document.getElementById ('teste') é escrito em linguagem JavaScript.

No entanto, como pode ser visto acima, as letras ficam muito juntas por não serem monoespaçadas e a solução para isso é colocá-las dentro de uma tag code.
Veja o exemplo abaixo.

O comando document.getElementById ('teste') é escrito em linguagem JavaScript.

Mais um exemplo abaixo em linguagem Ptyhon

num = int(imput ('digite um número')) if num % 2 == 0: print(f' O número{num} é par) else: print(f' O número {num} é ímpar) print('Fim do programa')

Porém da maneira utilizada acima, o código não coloca identação, ou seja, o código fica todo um ao lado do outro.

A maneira correta é colocar todo o conteúdo acima, inclusiva a tag code, dentro de outra tag chamada pre. Veja o exemplo abaixo

           
            num = int(imput ('digite um número'))
            if num % 2 == 0:
            print(f' O número{num} é par)
            else:
            print(f' O número {num} é ímpar)
            print('Fim do programa')
