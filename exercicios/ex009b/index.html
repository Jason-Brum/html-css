Usa-se a tag OL para listas ordenadas (ordered list) e a tag LI para cada item da lista

Números, letras e romanos

  1. Maçã
  2. Pera
  3. Melancia
  4. Banana
  5. Laranja
  6. Mamão

Usa-se o parâmetro type com a letra a minúscula ou A maiúscula para ser uma lista com letras em ordem alfabética

  1. Calabresa
  2. Marguerita
  3. Quatro queijos
  4. Portuguesa

Para algarismos romanos, basta usar o type I ou i para romanos maiúsculos ou minúsculos

Caso queira começar a lista em algum número ou letra que não seja 1 ou A, basta colocar o parâmetro start e o valor que você deseja iniciar.

Listas não ordenadas

  • Pão
  • Leite
  • Café
  • Extrato de tomate

Para listas que não necessitam de uma ordenação, utiliza-se a tag UL. Já para definir os sinais que vão aparecer, esolhese TYPE: circle, disc, square
